# Aripa Expert Consultancy Framework

Este directorio contiene la "inteligencia operativa" del proyecto Aripa. Son instrucciones, marcos de trabajo y checklists diseñados para que este asistente IA (u otros agentes futuros) mantenga la calidad técnica, el enfoque premium y la estrategia de crecimiento definida por **Javier Rodriguez de la Orden**.

## Estructura del Framework

1. **[Skills Profile](skills.md)**: Define la voz, el tono y las identidades expertas (SEO, GEO, Behavioral Economics, Dev, QA, Growth). Es la referencia maestra de comportamiento.
2. **Plugins (Módulos Especializados)**:
   - **[GEO/AEO Plugin](plugins/geo-aeo-plugin.md)**: Optimizaciones para motores de respuesta de IA.
   - **[Behavioral CRO Plugin](plugins/behavioral-cro-plugin.md)**: Psicología de la conversión.
   - **[Accessibility & QA Plugin](plugins/accessibility-qa-plugin.md)**: Excelencia técnica y ética.
   - **[Growth & Experimentation Plugin](plugins/growth-experimentation-plugin.md)**: Estrategia de escalado y tests A/B.

## Cómo usar este Framework

Para obtener los mejores resultados, recuerda al asistente que consulte estos archivos antes de realizar cambios estructurales o de contenido. Por ejemplo:

> *"Antes de redactar el nuevo servicio, consulta el `.expert/plugins/geo-aeo-plugin.md` para asegurar que el contenido sea citable por ChatGPT."*

O bien:

> *"Realiza una auditoría visual de la página basándote en el `.expert/plugins/accessibility-qa-plugin.md`."*
